수원 화성

경기도 수원시에 위치해 있는 수원 화성입니다.

화성은 경기도 수원에 있는 조선 시대의 성곽이다. 처음부터 계획하여 거주지로서의 읍성과 방어용 산성을 합하여 성곽도시로 만들었다. 전통적인 축성 기법에 과학적 기술을 활용하였으며, 이전의 성곽에 흔치 않았던 다양한 방어용 시설을 많이 첨가하였다. 주변 지형에 맞춰 자연스럽게 조성한 아름다움을 볼 수 있다

  • 구분 : 유네스코 세계유산
  • 국가 : 대한민국
  • 등재년도 : 1997년
